# GIT-PULL

## How to Use It?

- Download the script
- Create an alias (bash/zshell)

  ```Bash
    alias git-pull='/Users/roger-that/Documents/Codes/Shell_Script/Scripts/git-pull/git-pull.sh'
  ```

- Run the script inside a folder that has multiple repositories
